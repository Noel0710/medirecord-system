# MediRecord - Sistema de Recordatorio de Medicamentos

Sistema web para gestión y recordatorio de medicamentos para adultos mayores.

## Características
- Gestión de medicamentos y horarios
- Recordatorios por WhatsApp
- Perfiles para pacientes y cuidadores
- Historial de tomas
- Sistema de vinculación paciente-cuidador

## Tecnologías
- PHP 8+
- MySQL/MariaDB
- WhatsApp Business API
- HTML/CSS/JavaScript

## Instalación
1. Clonar repositorio
2. Importar `medirecord_db.sql`
3. Configurar `config.php`
4. Configurar WhatsApp Business API
